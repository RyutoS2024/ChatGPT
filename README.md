# ChatGPT

## セットアップ手順

1. 仮想環境を作成する

```bash
python3 -m venv env
```

2. 仮想環境を起動する

```bash
source env/bin/activate
```

3. 必要なパッケージをインストールする

```bash
pip install -r requirements.txt
```

4. 仮想環境を停止する

```bash
deactivate
```

## Whisper APIを使って音声ファイルをテキストファイルに変換する方法

1. API Keyを設定する
2. 音声ファイルをMP3フォルダにおく
3. テキストファイル用のMDフォルダを作成する
4. script.pyを実行する
